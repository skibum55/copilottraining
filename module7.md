## Module 7: Putting It All Together (3 minutes)

### 7.1 Complete Workflow Recap

You've learned a complete workflow for using Copilot effectively:

```
┌─────────────────────────────────────────────────────────────┐
│                 COPILOT WORKFLOW                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. SET UP CONTEXT                                          │
│     └─ Create prompts files with team standards            │
│     └─ Build library of reusable custom prompts            │
│     └─ Configure custom skills for common tasks            │
│                                                             │
│  2. USE AGENTS FOR AWARENESS                                │
│     └─ @workspace to understand codebase                   │
│     └─ @code for focused implementation                    │
│                                                             │
│  3. PLAN COMPLEX WORK                                       │
│     └─ "Create a phased plan. Ask questions first."        │
│                                                             │
│  4. EXECUTE WITH GUIDANCE                                   │
│     └─ Use standard phrases to control AI behavior         │
│     └─ Apply custom prompts for consistency                │
│     └─ Invoke skills for standardized tasks                │
│                                                             │
│  5. SUMMARIZE FOR HANDOFF                                   │
│     └─ "Summarize what we did and context for next phase"  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 7.2 Key Takeaways

1. **Prompts communicate intent** — be explicit about what you want
2. **Use standard phrases** to control AI behavior and collaboration style
3. **Agents provide context** — use @workspace for cross-file awareness
4. **Reusable prompts ensure consistency** — build a team library
5. **Custom skills standardize tasks** — create shortcuts for common operations
6. **Complex tasks need phases** — plan first, execute in steps, summarize for handoff
7. **Prompts files persist instructions** — onboard your AI to your project once
